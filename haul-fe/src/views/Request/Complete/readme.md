## 페이지 설명
예약이 완료된 후 나오는 화면입니다.<br/>
회원은 결제 후에, 비회원은 신청자 정보 입력 후에 접근할 수 있습니다. 

## 기능 설명
1. 홈으로 가기 버튼을 누르면 용달 종류 선택 화면으로 넘어갑니다.
2. 예약 확인하러 가기 버튼을 누르면 로그인 여부에 따라 맞는 예약 확인 페이지로 넘어갑니다.
