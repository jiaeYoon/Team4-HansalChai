## 페이지 설명
계산된 가격을 결제하는 화면입니다.<br />
로그인 한 회원만 화물 정보 입력 후 접근할 수 있습니다.

카드를 선택하고 결제를 합니다.

## 기능 설명
1. 등록된 카드 중 결제하기를 원하는 카드를 선택합니다.
2. 선택된 카드로 결제를 합니다.
