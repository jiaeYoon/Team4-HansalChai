## 페이지 설명
약관을 보여주는 페이지입니다.<br />


| 파라미터 | 타입 | 설명 |
|------|------|------|
| type | string | 보여 줄 약관 타입입니다. |

| type값 | 약관 |
|------|------|
| services | 이용약관 |
| privacy | 개인정보 처리방침 |
| transportation | 화물자동차 운송주선약관 |
| location | 위치기반서비스 이용약관 |
