이는 서비스에서 사용하는 내비게이션에 대한 컴포넌트입니다.

# Props

- selected - string
  - 선택된 항목을 전달하는 props
  - 가능한 값
    - “more” - 더보기
    - “reserv” (기본값) - 운송 예약
    - “check” - 예약 확인

# 예시

```html
<NavigationBar selected={"more"}></NavigationBar>
```
