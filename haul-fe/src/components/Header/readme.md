# Props

- back - boolean
  - 뒤로가기 버튼을 나타낼지 결정하는 props
  - 가능한 값
    - true(기본값)
    - falss
- home - boolean
  - 홈 버튼을 나타낼지 결정하는 props
  - 가능한 값
    - true
    - false(기본값)
- children - any
  - 헤더에 나타날 글을 나타낼 props
  - 가능한 값
    - Text
    - Typography

# 예시

```html
<header home="{true}" back="{true}">TrueTrue</header>
```
